# Concursos

omegaUp fornece recursos abrangentes de gerenciamento de concursos para organizar competições de programação.

## Tipos de concurso

- **Concursos Públicos**: Aberto a todos os usuários
- **Concursos Privados**: Restrito a participantes convidados
- **Concursos virtuais**: comece a qualquer momento em uma janela
- **Estilo USACO**: cronômetro individual por participante

## Recursos do concurso

- **Gerenciamento de problemas**: Adicionar/remover problemas, definir valores de pontos
- **Gerenciamento de participantes**: convide usuários, gerencie equipes
- **Placar**: classificações em tempo real com visibilidade configurável
- **Esclarecimentos**: Sistema de perguntas e respostas durante concursos
- **Sistema de penalidades**: cálculo de penalidade configurável
- **Decadência de pontuação**: redução de pontuação opcional com base no tempo
