# Contests

omegaUp provides comprehensive contest management capabilities for organizing programming competitions.

## Contest Types

- **Public Contests**: Open to all users
- **Private Contests**: Restricted to invited participants
- **Virtual Contests**: Start anytime within a window
- **USACO-style**: Individual timer per participant

## Contest Features

- **Problem Management**: Add/remove problems, set point values
- **Participant Management**: Invite users, manage teams
- **Scoreboard**: Real-time rankings with configurable visibility
- **Clarifications**: Q&A system during contests
- **Penalty System**: Configurable penalty calculation
- **Score Decay**: Optional time-based score decay
